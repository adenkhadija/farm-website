## Farm Website (Single Page Application):

A responsive single-page farm website built using HTML, CSS, and JavaScript. This project focuses on applying object-oriented programming (OOP) principles and dynamic DOM manipulation to create an interactive user experience.

### Tech Stack: 

- HTML – Structure
- CSS – Styling & layout
- JavaScript – Functionality, DOM manipulation, and OOP

### What I Learned: 
- How to structure a Single Page Application (SPA)
- Applying Object-Oriented Programming (OOP) concepts in JavaScript
- Manipulating and updating the DOM dynamically
- Writing cleaner, more maintainable front-end code
- Designing responsive layouts for better user experience

### Upcoming Improvements: 
- Add navigation with smooth scrolling or routing
- Add interactive forms (e.g., contact or newsletter signup)
